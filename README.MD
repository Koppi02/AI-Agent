# MAVERICK - Hierarchikus Hirdetés Klasszifikáló AI

Automatikusan kategorizálja a hirdetéseket hierarchikus rendszerben (Segment → Brand → BaseBrand → Advertiser) és képes felismerni új advertiser-eket OCR + NER kombinációval.

## 🚀 Funkciók

- ✅ Hierarchikus multi-task CNN (200K+ kép adatbázis)
- ✅ Új advertiser felismerés OCR + magyar NER-rel
- ✅ ABS_torveny.csv hierarchia validálás
- ✅ Google Colab támogatás GPU-val
- ✅ Checkpoint-alapú folytatható tanítás
